📊 Airline Disruption Analytics (AeroAnalytics)
🚀 Overview

This project analyzes airline operations data to identify flight delays, cancellations, and disruption patterns. It provides actionable insights using SQL, Python, and Power BI dashboards.

🎯 Objectives
Analyze flight delays and cancellations
Identify most disrupted airports
Track airline performance trends
Build interactive dashboard for decision-making
🛠️ Tech Stack
Python (Pandas, NumPy)
SQL (PostgreSQL/MySQL)
Power BI
Data Visualization
📈 Key Insights
Top 10 most disrupted airports
Delay trends across airlines
Seasonal patterns in cancellations
